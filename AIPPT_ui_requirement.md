> 本文档描述 AI 助手"子言"的 UI 交互需求，供 UI 设计师参考。
> 聚焦交互状态、视觉元素、弹窗浮层、反馈提示，不涉及业务逻辑细节。
> 业务逻辑详见 `AIPPT_requirement.md`。

---

## 第 1 章：页面结构与布局

### 1.1 AI 助手在编辑器中的位置

AI 助手由两部分组成：

| 组件 | 位置 | 说明 |
|------|------|------|
| AI 侧边栏 tab | 右侧工具栏竖向 tab 栏，紧挨"动画"tab 下方 | 与设计/样式/位置/动画 tab 并列 |
| 悬浮对话框 | 屏幕任意位置（可拖动） | 独立于编辑器布局，z-index 最高 |

### 1.2 右侧工具栏 tab 栏

竖向排列，宽 48px，每个 tab 上方图标 + 下方文字：

| Tab | 图标风格 | 文字 | 特殊样式 |
|-----|---------|------|----------|
| 设计/样式 | 线框 18px | 普通 10px | 选中时主题色 + 右侧 2px 边框 |
| 切换 | 线框 18px | 普通 10px | 同上 |
| 动画 | 线框 18px | 普通 10px | 同上 |
| **AI** | sparkles+十字星 线框 18px | **渐变色**（紫→蓝）10px 加粗 | 上方有 1px 分隔线，间距 4px |

## 第 2 章：交互状态清单

### 2.1 AI 侧边栏面板

#### 布局（从上到下）

1. **新建对话按钮** — 虚线边框，`+ 新建对话`
2. **历史记录列表** — 可滚动区域，占据中间空间
3. **快捷输入区** — 固定在底部，`border-top` 分隔

#### 快捷输入区状态

| 状态 | 输入框 | 语音按钮 | 发送按钮 | 提示文字 |
|------|--------|---------|---------|----------|
| 默认（未选中元素） | placeholder "给 AI 发指令..." | 麦克风图标 | 紫色，disabled | "✦ 输入指令，AI 为你服务" |
| 默认（已选中元素） | 同上 | 同上 | 同上 | "✦ 已选中元素，AI 将直接操作" |
| 输入中 | 有文字 | 同上 | 紫色，enabled | 同上 |
| 语音录制中 | 实时填入识别文字 | **红色方块 + 脉冲动画** | 根据内容 | 同上 |

#### 历史记录项状态

| 状态 | 背景 | 说明 |
|------|------|------|
| 默认 | 透明 | 显示图标 + 标题 + 消息数 + 时间 |
| hover | `#f3f4f6` | 浅灰背景 |
| 活跃（悬浮框打开中） | `#ede9fe` | 浅紫背景 |

图标区分：
- 普通对话 → 对话气泡图标
- 元素调整 → 编辑/画笔图标

#### 空状态

无历史记录时，中间区域显示：
```
暂无对话记录
在下方输入开始对话
```
居中，浅灰色，11px。

### 2.2 悬浮对话框

#### 整体属性

| 属性 | 值 | 说明 |
|------|-----|------|
| 宽度 | 400px | 固定 |
| 最大高度 | 680px | 内容不足时自适应 |
| 圆角 | 12px | 四角 |
| 阴影 | `0 8px 32px rgba(0,0,0,0.18)` | 双层阴影 |
| 边框 | 1px solid #e5e7eb | 浅灰 |

#### 标题栏

| 元素 | 说明 |
|------|------|
| 背景 | 渐变 `linear-gradient(135deg, #f5f3ff, #eff6ff)` |
| 左侧 | 对话气泡图标 + 会话标题（13px 500） |
| 右侧 | 关闭按钮（× 图标） |
| 交互 | 整个标题栏可拖动（cursor: move） |

#### 消息区域

| 消息类型 | 对齐 | 气泡样式 | 最大宽度 |
|----------|------|----------|----------|
| 用户消息 | 右对齐 | 蓝色背景 `#3b82f6`，白色文字 | 90% |
| AI 消息 | 左对齐 | 灰色背景 `#f3f4f6`，深色文字 | 90% |
| AI 思考中 | 左对齐 | 灰色背景，三个弹跳圆点 | — |

气泡圆角：10px。字号：12.5px。

**历史消息**：直接渲染，不走打字机动画。
**新消息**：AI 回复走打字机效果（3字符/10ms）。

#### 操作按钮

AI 回复下方的操作按钮：

| 按钮 | 样式 | 出现条件 |
|------|------|----------|
| 撤销 | 白底灰边框，11px | AI 执行了工具操作后 |
| 插入当前页 | 白底灰边框，11px | AI 生成了文本内容后 |
| 新建页插入 | 白底灰边框，11px | 同上 |

#### 工具栏

位于消息区和输入区之间：

| 元素 | 说明 |
|------|------|
| 工具按钮 | "🔧 工具"，hover 弹出菜单（向上展开） |
| 工具标签 | 选中工具后显示紫色标签 + × 关闭 |
| 绑定元素标识 | 右侧显示"✎ 绑定元素"（紫色） |

工具菜单：
- 向上展开，140px 宽
- 6 个选项：生成课堂引入 / 搜索背景知识 / 例题生成 / 互动环节设计 / 总结要点 / 生成演讲稿
- hover 150ms 延迟关闭（防间隙丢失）

#### 输入区

| 元素 | 说明 |
|------|------|
| 输入框 | 34px 高，6px 圆角，12px 字号 |
| 语音按钮 | 34×34px，白底灰边框，麦克风图标 |
| 发送按钮 | 34×34px，紫色背景，箭头图标 |

语音按钮录制状态：红色背景 + 方块停止图标 + 脉冲动画。

#### 欢迎状态

无消息时显示：
```
有什么需要帮忙的？
```
居中，浅灰色，13px，上下 30px padding。

## 第 3 章：视觉元素需求清单

### 3.1 Icon 需求

| 用途 | 当前实现 | 风格要求 |
|------|----------|----------|
| AI tab 图标 | SVG sparkles + 十字星 | 线框，1.5px 描边，18×18 |
| 对话气泡 | SVG message bubble | 线框，1.5px |
| 编辑/画笔 | SVG pencil | 线框，1.5px |
| 新建对话 + | SVG plus | 线框，2px |
| 关闭 × | SVG cross | 线框，2px |
| 发送箭头 | SVG paper plane | 线框，2px |
| 麦克风 | SVG microphone | 线框，2px |
| 停止录音 | SVG square (rounded) | 线框，2px |
| 工具扳手 | SVG wrench | 线框，2px |
| 下拉箭头 | SVG chevron-down | 线框，2px |

所有 icon 统一使用 SVG 内联，线框风格（Feather/Lucide 风格），不使用实体填充图标。

### 3.2 空状态图

| 场景 | 当前实现 | 需求 |
|------|----------|------|
| 无历史记录 | 纯文字"暂无对话记录" | 需补充：可考虑添加引导插图 |
| 悬浮框无消息 | 纯文字"有什么需要帮忙的？" | 需补充：可考虑添加 AI 助手形象图 |

### 3.3 颜色规范

| 用途 | 色值 | 说明 |
|------|------|------|
| AI 主题色 | `#8b5cf6` | 紫色，用于发送按钮、选中态、焦点态 |
| AI 渐变 | `#a78bfa → #60a5fa` | 紫→蓝，用于 tab 标签文字 |
| 用户气泡 | `#3b82f6` | 蓝色 |
| AI 气泡 | `#f3f4f6` | 浅灰 |
| 录音状态 | `#ef4444` | 红色 |
| 标题栏渐变 | `#f5f3ff → #eff6ff` | 浅紫→浅蓝 |

## 第 4 章：交互问题标注

### 4.1 已知交互问题

| 问题 | 当前状态 | 建议 |
|------|----------|------|
| 悬浮框无最小化 | 只能关闭 | 考虑添加最小化按钮，收起到角落 |
| 历史记录无删除 | 只能查看 | 需要左滑删除或右键菜单 |
| 历史记录无搜索 | 只能滚动浏览 | 记录多时需要搜索功能 |
| 工具选择无配置 | 选择后填入固定 prompt | 之前有高级配置面板（导入类型/难度等），可恢复 |
| 悬浮框位置不记忆 | 每次打开回到默认位置 | 可记住上次拖动位置 |

### 4.2 缺失的交互反馈

| 场景 | 缺失 | 建议 |
|------|------|------|
| 本地指令执行成功 | 无视觉反馈（元素闪烁等） | 被修改的元素短暂高亮 |
| 内容插入成功 | 仅文字回复 | 缩略图区域滚动到新页面 |
| 网络错误 | 仅文字提示 | 添加重试按钮 |
| 流式输出中断 | 显示部分内容 | 添加"继续生成"按钮 |

### 4.3 响应式适配

| 场景 | 当前处理 | 需求 |
|------|----------|------|
| 窄屏（<1200px） | 工具栏固定 310px | 需补充：考虑折叠侧边栏 |
| 移动端 | 有独立 Mobile 视图但无 AI | 需补充：移动端 AI 入口 |

## 第 5 章：组件复用说明

### 5.1 重复 UI 模式

| 模式 | 出现位置 | 复用建议 |
|------|----------|----------|
| 输入框 + 语音 + 发送 | 侧边栏底部、悬浮框底部 | 抽取为 `AIInputBar` 组件 |
| 操作按钮组 | 每条 AI 消息下方 | 抽取为 `ActionButtons` 组件 |
| 工具选择菜单 | 悬浮框工具栏 | 可复用 Popover 组件 |

### 5.2 与编辑器已有组件的关系

| AI 组件 | 编辑器已有组件 | 关系 |
|---------|---------------|------|
| 悬浮对话框 | `MoveablePanel.vue` | 可参考其拖动逻辑 |
| 工具菜单 | `Popover.vue` | 可直接复用 |
| 消息列表滚动 | 编辑器各面板 | 使用相同的 `overflow-overlay` mixin |
