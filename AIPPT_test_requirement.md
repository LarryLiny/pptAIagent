## 第 3 章：状态流转测试

### 3.1 消息发送状态机

| 当前状态 | 操作 | 预期下一状态 | 测试要点 |
|----------|------|-------------|----------|
| 空闲 | 输入文字 + 回车 | 流式输出 | 发送按钮变 disabled |
| 空闲 | 输入为空 + 回车 | 空闲 | 不发送，无反应 |
| 流式输出 | 等待完成 | 完成 | 显示操作按钮 |
| 流式输出 | 点击发送 | 流式输出 | 按钮 disabled，无法发送 |
| 完成 | 输入新消息 + 回车 | 流式输出 | 新一轮对话 |

### 3.2 悬浮框状态

| 当前状态 | 操作 | 预期下一状态 |
|----------|------|-------------|
| 关闭 | 点击"新建对话" | 打开（空会话） |
| 关闭 | 点击历史记录 | 打开（加载历史消息） |
| 打开 | 点击关闭按钮 | 关闭（会话保留） |
| 打开 | 拖动标题栏 | 打开（位置改变） |

### 3.3 本地指令 vs LLM 路由

| 条件 | 输入 | 预期路由 |
|------|------|----------|
| 选中文本框 | "字号改成30" | 本地执行（<50ms） |
| 选中文本框 | "把这段话改写得更简洁" | LLM Function Calling |
| 未选中元素 | "字号改成30" | LLM（无选中元素，本地不处理） |
| 未选中元素 | "帮我生成一段课堂引入" | LLM 文本回复 |
| 选中文本框 | "改成红色" | 本地执行 |
| 选中文本框 | "帮我找几张图片" | LLM search_images |

## 第 5 章：数据边界测试

### 5.1 输入字段

| 字段 | 正常值 | 边界值 | 异常值 | 预期 |
|------|--------|--------|--------|------|
| 消息输入 | "你好" | 1 个字符 "a" | 空字符串 "" | 空字符串不发送 |
| 消息输入 | 50 字 | 1000 字 | 5000 字 | 均可发送，长消息正常显示 |
| 字号指令 | "字号改成30" | "字号改成8" / "字号改成200" | "字号改成0" / "字号改成999" | 8-200 范围内执行，超出范围不执行 |
| 颜色指令 | "改成红色" | "#fff" / "#000000" | "改成彩虹色" | 已知颜色名执行，未知颜色走 LLM |
| 图片搜索 | "classroom" | "a" | "" | 均调用 API，空关键词不搜索 |

### 5.2 会话数据

| 场景 | 测试数据 | 预期 |
|------|---------|------|
| 会话标题截取 | 用户首条消息 21 字 | 标题显示前 20 字 + "..." |
| 会话标题截取 | 用户首条消息 5 字 | 标题显示完整 5 字 |
| sessionStorage 容量 | 100 个会话 × 50 条消息 | 正常存储（sessionStorage 通常 5MB） |
